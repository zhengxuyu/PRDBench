# Intelligent Set-Meal Management System

A command-line tool for food & beverage merchants to manage set-meal lifecycle including ingredient management, cost calculation, intelligent combo generation, and evaluation.

## Setup

```bash
pip install -r requirements.txt
```

## Usage

```bash
python main.py
```

## Project Structure

- `main.py` - Main entry point with menu system
- `data_manager.py` - Ingredient and extra cost data management
- `cost_calculator.py` - Cost calculation module
- `combo_generator.py` - Intelligent combo generation
- `combo_evaluator.py` - Combo evaluation and comparison
- `data/` - Data storage directory
