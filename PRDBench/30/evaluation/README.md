## Test Plan Overview

The test plan contains 17 test points, divided into three types:

### 1. Shell Interaction Tests (12 items)
- 1.1 Program startup and help information
- 2.1.2 Data integrity verification and exception handling
- 3.1 TransE model training startup
- 3.2 Training parameter configuration
- 3.4 Loss value calculation and monitoring (20 epochs, verify convergence trend)
- 4.2 Evaluation metric calculation - MeanRank
- 4.3 Evaluation metric calculation - Hit@10
- 5.1a Entity vector file output
- 5.1b Relation vector file output
- 5.3a Evaluation report generation - File format
- 5.3b Evaluation report content - Including key metrics
- 6.1 Error handling and prompts

### 2. File Comparison Tests (4 items)
- 2.1.1a Data loading - Entity mapping file generation
- 2.1.1b Data loading - Relation mapping file generation
- 4.1 Link prediction function (using gold standard dataset)
- 5.2 Loss curve visualization

### 3. Unit Tests (1 item)
- 3.3 Vector initialization verification

## Main Components

### 1. Main Program (src/main.py)
Created a unified command-line interface that supports:
- `preprocess`: Data preprocessing
- `train`: Model training (all output files saved to specified directory)
- `evaluate`: Model evaluation

### 2. Test Data
- **Valid Data**: FB15K-237 dataset in src/data/
- **Gold Standard Data**: Small deterministic dataset in evaluation/golden_data/, ensuring evaluation can produce valid results
- **Invalid Data**: Test data with illegal entities in evaluation/invalid_data/

### 3. Expected Output Files
Created corresponding expected output files for each file_comparison type test to verify the file format and content generated by the program. Expected files are consistent with actual data files to ensure test accuracy.

### 4. Unit Tests
Created pytest-compatible unit tests to verify the correctness of vector initialization.

## Usage

### Run Shell Interaction Tests
```bash
python src/main.py --help
python src/main.py train --input ./src/data --output ./test_output --epochs 2
python src/main.py evaluate --input ./src/data --model ./test_output
```

### Run File Comparison Tests
After executing the corresponding command, compare the generated file with the expected file:
```bash
python src/main.py preprocess --input ./src/data --output ./test_output
# Compare test_output/entity2id.txt with evaluation/expected_entity2id.txt
```

### Run Unit Tests
```bash
pytest evaluation/tests/test_vector_initialization.py::test_vector_dimensions
```
